### 文件说明 Documentation
* 该文件数据包括了海外地区首例确诊时间、输出地、输入地及其经纬度。
* 输出地输入地以国内媒体报道的各国官方通报首例确诊病例来源为标准。
* 伊朗、埃及这些本国官方未交代清楚来源的我们都写了来源不明。
